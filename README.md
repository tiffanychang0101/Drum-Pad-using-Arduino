## 介紹
以義大利的村莊 Alberobello 為原型  
製作成的節奏樂器  

Alberobello的特色是屋頂用石塊堆成圓錐形  
牆壁用石灰塗成白色  
據說是當地人為了逃稅而建的能立馬拆掉的建築物  

![5](https://github.com/tiffanychang0101/Drum-Pad-using-Arduino/blob/master/images/IMAG0189.jpg)
